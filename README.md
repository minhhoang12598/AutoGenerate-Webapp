# Pikee Webapp

Cấu trúc dự án cho Pikee-webapp.

Thư mục:
- `backend/` - mã nguồn backend
  - `app/core/` - lõi ứng dụng
  - `app/api/` - route/endpoint
  - `app/services/` - services/logic
  - `tests/` - unit/integration tests
- `frontend/` - mã nguồn frontend
- `storage/` - lưu ảnh job (được ignore)
- `scripts/` - tool và script tiện ích
